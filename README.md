This repository contains a React-based portfolio project built to satisfy the assignment instruction:

- Convert Activity 1 output to React-based project

## Local setup

1. `npm install`
2. `npm run dev`
3. Open `http://localhost:5173`

## Build

- `npm run build`
- `npm run preview`

## Project content

- `src/App.jsx`: portfolio content with About, Skills, Projects, Reflection
